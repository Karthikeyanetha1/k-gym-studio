## 📌 Overview

K Gym Studio is a high-impact fitness website built as a **CodeWithK** freelance demo. Designed in a dark editorial style with red accents, it presents a complete gym brand — programs, trainers, gallery, pricing, and booking — in a single production-ready HTML file.

> 🔗 **Live:** [k-gym-studio.vercel.app](https://k-gym-studio.vercel.app)

---

## ✨ Features

| # | Feature | Description |
|:---:|---|---|
| 🏋️ | **Hero Section** | Full-viewport animated gym illustration with stats |
| 📢 | **Scrolling Ticker** | Animated marquee of program names |
| 🏃 | **6 Programs** | Strength, HIIT, Boxing, Yoga, Cardio, Nutrition |
| 👥 | **Trainer Profiles** | 3 expert trainers with certs & bios |
| 📊 | **Stats Band** | Members, trainers, satisfaction, years — animated |
| 🖼️ | **Facility Gallery** | Weight floor, cardio deck, boxing ring, yoga studio |
| 💳 | **3-Tier Pricing** | Basic / Pro / Elite with features list |
| ⭐ | **Testimonials** | 3 verified member reviews |
| 📬 | **Booking Form** | Free trial inquiry form with EmailJS |
| 🗺️ | **Live Map Pin** | Animated SVG location map for HITEC City |
| 📱 | **Fully Responsive** | Mobile, tablet, desktop |

---

## 🛠️ Tech Stack

```
Frontend     →   HTML5, CSS3, Vanilla JavaScript
Animations   →   Pure CSS keyframes — fadeUp, slideIn, float, pulse, spin
Icons        →   Inline SVG — zero dependencies
Illustrations→   Custom embedded SVGs (hero, programs, trainers, gallery)
Email        →   EmailJS SDK — real form submissions
Fonts        →   Google Fonts — Barlow Condensed + Barlow
Hosting      →   Vercel (auto-deploy from GitHub)
```
